PETUNJUK SIMULASI

1. Tujuan

Simulasi ini digunakan untuk mempelajari perkembangan model atom dari sederhana hingga modern secara visual dan interaktif.

2. Model yang Tersedia

  • Model Atom Dalton
  • Model Atom Thomson
  • Model Atom Rutherford
  • Model Atom Bohr
  • Model Atom Mekanika Kuantum

3. Cara Menggunakan

  1. Pilih Model Atom
    Gunakan tombol atau menu untuk memilih model atom.
  2. Putar Model 3D
    Klik dan geser (drag) untuk melihat dari berbagai sisi. Gunakan scroll untuk zoom.
  3. Baca Penjelasan
    Deskripsi model tersedia di bagian bawah layar.
  4. Klik Partikel Atom
    Klik proton, neutron, atau elektron untuk melihat informasinya.

4. Tips

  • Gunakan urutan dari model sederhana ke modern.
  • Amati perbedaan setiap model atom.
  • Klik semua bagian atom untuk memahami fungsinya.

5. Penutup

Simulasi ini membantu memahami struktur dan perkembangan model atom melalui tampilan 3D interaktif.

PETUNJUK SIMULASI

1. Tujuan

Simulasi ini membantu memahami struktur atom melalui notasi atom, jumlah partikel, dan penyusunan partikel dalam orbital secara interaktif.

2. Mode Simulasi

Sebelum memulai, pilih salah satu mode berikut:

Mode Notasi Atom
  • Isi nomor atom, nomor massa, dan muatan.
  • Jumlah proton, neutron, dan elektron akan menyesuaikan otomatis.

Mode Partikel
  • Isi jumlah proton, neutron, dan elektron secara manual.
  • Notasi atom akan terbentuk otomatis.

Mode Struktur Atom
  • Gunakan partikel yang tersedia untuk menyusun atom.
  • Pindahkan proton, neutron, dan elektron ke dalam orbital.

3. Cara Menggunakan

  1. Pilih Mode
    Pilih salah satu mode sebelum memulai simulasi.
  2. Isi Data (Sesuai Mode)
    Masukkan data sesuai dengan mode yang dipilih.
  3. Susun Struktur Atom (Mode Struktur)
    • Terdapat 3 wadah (mangkok): proton, neutron, dan elektron.
    • Seret (drag) partikel ke dalam orbital atom.
  4. Amati Perubahan
    Perhatikan perubahan struktur atom, jumlah partikel, dan notasi atom.

4. Aturan Penting

  • Hanya fitur sesuai mode yang dapat digunakan.
  • Mode notasi: hanya isi notasi atom.
  • Mode partikel: hanya isi jumlah partikel.
  • Mode struktur: hanya menyusun partikel ke orbital.

5. Tips

  • Gunakan mode notasi untuk memahami simbol atom.
  • Gunakan mode partikel untuk latihan menghitung.
  • Gunakan mode struktur untuk memahami posisi partikel dalam atom.

6. Penutup

Simulasi ini mempermudah pemahaman struktur atom secara visual, interaktif, dan menyenangkan.
