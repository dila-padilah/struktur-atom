PETUNJUK PENGGUNAAN

✨ 📘 Petunjuk Penggunaan Website Pembelajaran Interaktif

Selamat datang di website pembelajaran interaktif ✨
Media ini dirancang untuk membantu kamu memahami konsep kimia secara visual, eksploratif, dan mandiri melalui materi, simulasi, dan evaluasi yang terintegrasi. Agar pengalaman belajar lebih optimal, ikuti panduan berikut 👇


🚀 Jelajahi Menu Utama

Pada halaman utama, kamu akan menemukan beberapa fitur utama:

  • 🎯 Capaian Pembelajaran → target yang harus kamu kuasai
  • 📚 Materi Pembelajaran → penjelasan konsep
  • 🧪 Simulasi Interaktif → eksplorasi visual
  • 📝 Evaluasi → uji pemahaman

💡 Pilih menu sesuai tahap belajarmu.


🎯 Capaian Pembelajaran

Fitur ini membantu kamu memahami tujuan belajar sebelum memulai. Terdiri dari:

  • Pemahaman Kimia → memahami konsep dasar
  • Keterampilan Proses → melatih cara berpikir ilmiah

✨ Dengan membaca bagian ini, kamu jadi tahu “habis belajar ini, aku harus bisa apa?”


📚 Materi Pembelajaran

Di sini kamu akan mempelajari konsep secara bertahap dengan bantuan:

  • Penjelasan teks yang jelas
  • Ilustrasi dan animasi
  • Contoh soal

Cara belajar yang disarankan:

  1. Pilih materi yang ingin dipelajari
  2. Pahami penjelasan dan visualisasi
  3. Amati contoh soal
  4. Kerjakan latihan di akhir materi

💡 Tips: Jangan hanya membaca — coba pahami alurnya!


🧪 Simulasi Interaktif

Inilah bagian paling seru! Kamu bisa belajar dengan langsung berinteraksi 🎉 Terdapat dua jenis simulasi:

🔬 Model Atom

  • Menampilkan perkembangan model atom dari masa ke masa
  • Membantu memahami bentuk atom secara visual

⚛️ Struktur Kimia

  • Menunjukkan hubungan antara proton, elektron, dan neutron
  • Membantu memahami bagaimana struktur atom terbentuk

📝 Evaluasi

Setelah belajar, saatnya menguji pemahamanmu!

Ciri-ciri evaluasi:

  • Soal lebih menantang
  • Mengukur pemahaman secara keseluruhan
  • Menampilkan skor akhir

Cara mengerjakan:

  1. Masuk ke menu evaluasi
  2. Jawab semua soal dengan teliti
  3. Lihat hasil akhir yang ditampilkan sistem

🎯 Anggap ini sebagai “level akhir” dari pembelajaranmu!


🖱️ Interaksi dalam Website

Website ini dirancang interaktif, jadi kamu bisa:

  • Klik objek untuk melihat informasi
  • Mengisi data pada kolom tertentu
  • Drag & drop untuk mencocokkan jawaban
  • Melihat popup penjelasan secara langsung

✨ Semakin sering kamu mencoba, semakin paham!


💡 Tips Agar Belajar Lebih Maksimal

  • Gunakan laptop/PC untuk tampilan terbaik
  • Ikuti alur belajar secara berurutan
  • Jangan skip simulasi (ini kunci pemahaman!)

📌 Alur terbaik:

Capaian Pembelajaran → Materi → Simulasi → Evaluasi


🌟 Penutup

Website ini tidak hanya untuk membaca, tapi untuk mengalami proses belajar secara langsung.
